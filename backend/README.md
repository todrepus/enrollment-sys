# 소개
학생, 교수, 강의실 등 학교 자원들을 관리하는 기능과
수강신청을 할 수 있는 기능을 제공하는 학사정보시스템을 개발하고있습니다.
단순화된 형태의 시스템 입니다.

# 기술 스택
* Java 18
* Vue.js
* Spring Boot 3
* Spring FrameWork
* Html / CSS / JavaScript
